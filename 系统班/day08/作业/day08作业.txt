一. 代码分析5题：
	第一个：分析如下代码，是否有错, 若有错判断哪行报错，并说出报错原因
		class Demo{
			final int a = 10; 
		}

		class Test{
			public static void main(String[] args) {
				Demo d = new Demo(); 
				d = new Demo(); 
				d.a = 20; //出错：
			}
		}

答：fianl修饰a即a的值不可更改


	第二个：分析如下代码,是否有错, 若有错判断哪行报错，并说出报错原因
		class Demo{
			int a = 10; 
		}

		class Test{
			public static void main(String[] args) {
				final Demo d = new Demo();
				d = new Demo();//出错 
				d.a = 20; 
			}
		}
  

答：final修饰的引用类型变量的地址值不可更改
